# Chaitanya Sanagana — Generative AI Engineer Portfolio

A modern, recruiter-focused portfolio built with Next.js, TypeScript, and Tailwind CSS to showcase my work in Generative AI, Retrieval-Augmented Generation (RAG), document intelligence, semantic retrieval, and compliance-aware backend systems.

This portfolio is designed to present my profile as a Generative AI Engineer with experience building AI-powered applications for regulated and enterprise-style environments, with a strong focus on practical systems, product thinking, and clean technical execution.

## About This Portfolio

The portfolio highlights my work across:

- Generative AI and LLM applications
- Retrieval-Augmented Generation (RAG)
- Document intelligence workflows

- Backend engineering with Python and FastAPI
